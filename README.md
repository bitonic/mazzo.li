## Building

To build the Hakyll website (requires [Nix](https://nixos.org/)):

```
$ # Builds and exists
$ make hakyll-build
$ # Builds continuously
$ make hakyll-build
```

To build the JavaScript scripts:

```
$ make yarn-install
$ # Build dev build
$ make webpack-build
$ # Build dev build continuosly
$ make webpack-watch
$ # Build prod build
$ make webpack-build-prod
```
